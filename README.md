# Python, Data Analysis, Machine Learning & Deep Learning

A hands-on learning repository covering **Python programming**, **Data Analysis**, **Machine Learning**, and **Deep Learning** through structured Jupyter notebooks and mini-projects.  
This repository demonstrates practical implementation of core concepts and is designed for **skill development and portfolio showcase**.

---

## Skills & Concepts Covered

### Python
- Core Python syntax and data structures  
- Functions, loops, and conditionals  
- File handling and exception handling  

### Data Analysis
- NumPy for numerical computation  
- Pandas for data manipulation  
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization using Matplotlib & Seaborn  

### Machine Learning
- Supervised Learning  
  - Linear Regression  
  - Logistic Regression  
  - Decision Trees  
  - K-Nearest Neighbors (KNN)  
- Unsupervised Learning  
  - K-Means Clustering  
  - Hierarchical Clustering  
- Feature scaling and model evaluation  

### Deep Learning
- Neural network fundamentals  
- Artificial Neural Networks (ANN)  
- Convolutional Neural Networks (CNN) basics  
- Model implementation using TensorFlow & Keras

## Setup & Installation

### 1. Pre-Requisites:
- Python 3.10 or above
- Git (Optional for cloning the repository)

### 2. Clone the repository

If you haven't configured ssh keys, use the HTTPS option.

**HTTPS**
```bash
git clone https://github.com/SusmitaTripathi/Data-Science-ML-AI.git
```

**SSH**
```bash
git clone git@github.com:SusmitaTripathi/Data-Science-ML-AI.git
```

```bash
cd Data-Science-ML-AI
```

### 3. Create and activate a virtual environment

#### Create virtual environment
```bash
python -m venv .venv
```

#### Activate virtual environment
**Windows**
```bash
.\.venv\Scripts\activate
```
**Mac\Linux**
```bash
source .venv/bin/activate
```
### 4. Installing Dependencies
```bash
pip install -r requirements.txt
```

### 5. Launch Jupyter Notebooks
```bash
jupyter notebook
``` 
or
```bash
jupyter lab
```


### 6. Run the notebooks


Open the notebooks and execute the cells sequentially.

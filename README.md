# Python, Data Analysis, Machine Learning & Deep Learning

A hands-on learning repository covering **Python programming**, **Data Analysis**, **Machine Learning**, and **Deep Learning** through notebooks and mini-projects.  
This repository demonstrates practical implementation of concepts and is intended for **skill building and portfolio showcase**.

---

## Skills & Concepts Covered

### Python
- Core Python syntax and data structures  
- Functions, loops, and conditionals  
- File handling and exception handling  

### Data Analysis
- NumPy for numerical computation  
- Pandas for data manipulation  
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization using Matplotlib & Seaborn  

### Machine Learning
- Supervised Learning  
  - Linear Regression  
  - Logistic Regression  
  - Decision Trees  
  - KNN  
- Unsupervised Learning  
  - K-Means Clustering  
  - Hierarchical Clustering  
- Feature scaling and model evaluation  

### Deep Learning
- Neural network fundamentals  
- Artificial Neural Networks (ANN)  
- Convolutional Neural Networks (CNN) basics  
- Model implementation using TensorFlow & Keras
